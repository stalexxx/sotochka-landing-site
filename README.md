# SOTOCHKA landing

Отдельный лендинг для курса Stepik:
https://stepik.org/course/92015/promo

## Файлы
- `index.html` — структура страницы
- `styles.css` — стили лендинга

## Как открыть
Просто открой `index.html` в браузере.

- блок с преимуществами
- программа курса
- блок про автора
- FAQ
- CTA на Stepik
